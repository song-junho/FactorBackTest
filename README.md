# FactorBackTest
> InvestStrategy 의 output인 투자스케줄을 기반으로 백테스트 진행  
> Output: sheet_balance(자산) , sheet_book(북) , sheet_trade(거래)

1. sheet_balance
   * 월별 전체 자산 , 투자 자산 , 현금 자산 기록
2. sheet_book
   * 월별 보유 자산 현황
3. sheet_trade
   * 거래 기록 

# OutPut Sample
 * item_type
   * 의미: 자산 타입 
   * 종류: stock, bond, commodity  
 * w_type
   * 의미: weight 기준
   * 종류: equal, market_cap, z_score  
 * weight
   * 의미: 투자비중
